# vSphere的NSX-T Data Center安装流程

遵序建议的过程顺序。

1. 查看NSX Manager 安装要求。
2. 配置必要的端口和协议。
3. 安装NSX Manager
4. 登录到新创建的NSX Manager
5. 配置计算管理器
6. 部署其他NSX Manager设备以形成集群。
7. 查看NSX Edge安装要求。
8. 安装NSX Edge。
9. 创建NSX Edge集群。
10. 创建传输区域。
11. 创建主机传输节点。
